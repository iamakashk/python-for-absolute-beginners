# Loops in Python — Detailed Guide with Examples

## What is a Loop?

A loop is used when we want to execute a block of code multiple times.

Instead of writing the same code again and again, loops help us automate repetitive tasks.

---

# Real-Life Example

Imagine you are teaching 100 students.

Without loops:

```python
print("Welcome")
print("Welcome")
print("Welcome")
```

This becomes repetitive.

With loops:

```python
for i in range(100):
    print("Welcome")
```

This is shorter, cleaner, and easier to maintain.

---

# Types of Loops in Python

Python mainly provides:

- `for` loop
- `while` loop

---

# 1. for Loop in Python

A `for` loop is used to iterate over a sequence.

A sequence can be:

- list
- tuple
- string
- set
- dictionary
- range

---

## Syntax of for Loop

```python
for variable in sequence:
    # code block
```

---

# Example 1 — Basic for Loop

```python
for i in range(5):
    print(i)
```

## Output

```python
0
1
2
3
4
```

---

## Explanation

`range(5)` generates numbers from `0` to `4`.

### Loop Execution

| Iteration | Value of i |
|---|---|
| 1 | 0 |
| 2 | 1 |
| 3 | 2 |
| 4 | 3 |
| 5 | 4 |

---

# Understanding range()

`range()` is commonly used with loops.

---

## range(stop)

```python
for i in range(5):
    print(i)
```

### Output

```python
0 1 2 3 4
```

---

## range(start, stop)

```python
for i in range(1, 6):
    print(i)
```

### Output

```python
1 2 3 4 5
```

---

## range(start, stop, step)

```python
for i in range(1, 11, 2):
    print(i)
```

### Output

```python
1 3 5 7 9
```

The loop picks one item at a time from the list.

# Looping Through a Dictionary

# Access Keys

```python
student = {
    "name": "Akash",
    "age": 25,
    "city": "Mumbai"
}

for key in student:
    print(key)
```

---

# Access Values

```python
for value in student.values():
    print(value)
```

---

# Access Both Key and Value

```python
for key, value in student.items():
    print(key, value)
```

# 2. while Loop in Python

A `while` loop runs as long as the condition is `True`.

---

## Syntax of while Loop

```python
while condition:
    # code block
```

---

# Example of while Loop

```python
count = 1

while count <= 5:
    print(count)
    count += 1
```

## Output

```python
1
2
3
4
5
```

---

# Explanation

## Initial Value

```python
count = 1
```

---

## Condition Check

```python
count <= 5
```

If condition becomes `False`, loop stops.

---

## Increment

```python
count += 1
```

Without increment, loop becomes infinite.

---

# Infinite Loop

```python
while True:
    print("Hello")
```

This loop never stops.

Use carefully.

---

# break Statement

`break` is used to immediately stop the loop.

---

## Example

```python
for i in range(10):

    if i == 5:
        break

    print(i)
```

## Output

```python
0
1
2
3
4
```

---

## Explanation

When `i` becomes `5`, loop stops completely.

---

# continue Statement

`continue` skips the current iteration.

---

## Example

```python
for i in range(5):

    if i == 2:
        continue

    print(i)
```

## Output

```python
0
1
3
4
```

---

## Explanation

When `i` is `2`, that iteration is skipped.

---

# pass Statement

`pass` does nothing.

It is used as a placeholder.

---

## Example

```python
for i in range(5):
    pass
```

---

# Nested Loops

A loop inside another loop is called a nested loop.

---

## Example

```python
for i in range(3):

    for j in range(2):
        print(i, j)
```

## Output

```python
0 0
0 1
1 0
1 1
2 0
2 1
```

---

# else with Loops

Python allows `else` with loops.

The `else` block executes when loop finishes normally.

---

## Example

```python
for i in range(5):
    print(i)
else:
    print("Loop finished")
```

---

# Example with break

```python
for i in range(5):

    if i == 3:
        break

    print(i)

else:
    print("Loop finished")
```

Here, `else` will NOT execute because loop stopped using `break`.

# Common Mistakes in Loops

# 1. Infinite Loop

```python
count = 1

while count <= 5:
    print(count)
```

## Problem

`count` is never incremented.

---

# 2. Wrong Indentation

## Wrong

```python
for i in range(5):
print(i)
```

This causes:

```python
IndentationError
```

---

## Correct

```python
for i in range(5):
    print(i)
```

---

# Difference Between for and while Loop

| for Loop | while Loop |
|---|---|
| Used when number of iterations is known | Used when iterations are unknown |
| Easier syntax | More flexible |
| Mostly used with sequences | Mostly used with conditions |
