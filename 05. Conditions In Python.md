# Python Conditions Complete Guide

# Introduction

Conditions in Python are used to make decisions in a program.

A program can:

- Execute one block of code
- Skip another block
- Make decisions based on `True` or `False` conditions

Conditions are one of the most important concepts in programming.

---

# What is a Condition?

A condition is an expression that returns:

- `True`
- `False`

## Example

```python
10 > 5
```

## Output

```python
True
```

---

# if Statement

The `if` statement executes code only when the condition is `True`.

---

## Syntax

```python
if condition:
    code
```

---

## Example

```python
age = 18

if age >= 18:
    print("You can vote")
```

---

# Indentation in Python

Python uses indentation instead of curly braces.

---

## Correct

```python
if True:
    print("Hello")
```

---

## Wrong

```python
if True:
print("Hello")
```

---

# if-else Statement

Used when you want one block to run if condition is `True` and another block if condition is `False`.

---

## Syntax

```python
if condition:
    code
else:
    code
```

---

## Example

```python
age = 16

if age >= 18:
    print("Eligible")
else:
    print("Not Eligible")
```

---

# if-elif-else Statement

Used when checking multiple conditions.

---

## Syntax

```python
if condition1:
    code
elif condition2:
    code
else:
    code
```

---

## Example

```python
marks = 75

if marks >= 90:
    print("Grade A")
elif marks >= 70:
    print("Grade B")
elif marks >= 50:
    print("Grade C")
else:
    print("Fail")
```

---

# Multiple elif Example

```python
number = 0

if number > 0:
    print("Positive")
elif number < 0:
    print("Negative")
else:
    print("Zero")
```

---

# Nested if Statement

An `if` inside another `if` is called nested `if`.

---

## Example

```python
age = 25
citizen = True

if age >= 18:
    if citizen:
        print("Eligible to vote")
```

---

# Comparison Operators

Comparison operators are used in conditions.

| Operator | Meaning |
|---|---|
| `==` | Equal to |
| `!=` | Not equal to |
| `>` | Greater than |
| `<` | Less than |
| `>=` | Greater than OR equal |
| `<=` | Less than or equal |

---

# Examples

```python
print(10 == 10)
print(10 != 5)
print(20 > 10)
print(5 < 3)
```

---

# Logical Operators

Logical operators combine multiple conditions.

| Operator | Meaning |
|---|---|
| `and` | Both conditions must be True |
| `or` | At least one condition must be True |
| `not` | Reverses the condition |

---

# and Operator

## Example

```python
age = 20
citizen = True

if age >= 18 and citizen:
    print("Eligible")
```

---

# or Operator

## Example

```python
username = "admin"
password = "1234"

if username == "admin" or password == "admin":
    print("Access Granted")
```

---

# not Operator

## Example

```python
is_logged_in = False

if not is_logged_in:
    print("Please login")
```

---

# Short Hand if

## Syntax

```python
if condition: statement
```

---

## Example

```python
age = 20

if age >= 18: print("Adult")
```

---

# Short Hand if-else (Ternary Operator)

## Syntax

```python
value_if_true if condition else value_if_false
```

---

## Example

```python
age = 17

result = "Adult" if age >= 18 else "Minor"

print(result)
```

---

# Truthy and Falsy Values

Python treats some values as `False` automatically.

---

# Falsy Values

- `False`
- `0`
- `None`
- `""`
- `[]`
- `{}`
- `set()`

Everything else is generally treated as `True`.

---

## Example

```python
name = ""

if name:
    print("Name exists")
else:
    print("Name is empty")
```

---

# Checking Membership

Using `in` and `not in`.

---

## Example Using in

```python
fruits = ["apple", "banana", "mango"]

if "apple" in fruits:
    print("Apple found")
```

---

## Example Using not in

```python
if "grapes" not in fruits:
    print("Grapes not found")
```

---

# Pass Statement

Used when a block is required but no code is written yet.

---

## Example

```python
age = 18

if age > 18:
    pass
```

---

# Real World Examples

# Login System Example

```python
username = "akash"
password = "1234"

if username == "akash" and password == "1234":
    print("Login Successful")
else:
    print("Invalid Credentials")
```

---

# ATM Withdrawal Example

```python
balance = 5000
withdraw = 2000

if withdraw <= balance:
    print("Withdrawal Successful")
else:
    print("Insufficient Balance")
```

---

# Even Odd Program

```python
number = 10

if number % 2 == 0:
    print("Even")
else:
    print("Odd")
```

---

# Positive Negative Number

```python
number = -5

if number > 0:
    print("Positive")
elif number < 0:
    print("Negative")
else:
    print("Zero")
```

---

# Biggest of Two Numbers

```python
a = 10
b = 20

if a > b:
    print("a is bigger")
else:
    print("b is bigger")
```

---

# Biggest of Three Numbers

```python
a = 10
b = 50
c = 30

if a > b and a > c:
    print("a is biggest")
elif b > a and b > c:
    print("b is biggest")
else:
    print("c is biggest")
```

---

# Important Points

- Conditions return `True` or `False`
- Python uses indentation
- `if` is used for decision making
- `elif` checks multiple conditions
- `else` runs when all conditions fail
- Logical operators combine conditions
- Nested `if` means `if` inside another `if`

---

# Common Mistakes

# Missing Colon

## Wrong

```python
if age > 18
    print("Adult")
```

---

## Correct

```python
if age > 18:
    print("Adult")
```

---

# Wrong Indentation

## Wrong

```python
if True:
print("Hello")
```

---

## Correct

```python
if True:
    print("Hello")
```

---

# Using = Instead of ==

## Wrong

```python
if age = 18:
```

---

## Correct

```python
if age == 18:
```
