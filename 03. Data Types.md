# Python Data Types Complete Guide

# What is a Data Type?

A data type defines:

- What kind of value a variable stores
- What operations can be performed on it
- How Python manages the value internally

Example:

```python
age = 25
```

Here:

- `25` is an integer value
- Its data type is `int`

---

# Why Are Data Types Important?

Data types help Python understand:

- How much memory to allocate
- What operations are valid
- How data should behave

Example:

```python
10 + 20
```

works because both are numbers.

But:

```python
"10" + "20"
```

performs string concatenation.

---

# Categories of Python Data Types

Python data types are mainly divided into:

1. Numeric Types
2. Sequence Types
3. Set Types
4. Mapping Type
5. Boolean Type
6. Binary Types
7. None Type

---

# 1. Numeric Data Types

Numeric data types store numbers.

## Types

- `int`
- `float`
- `complex`

---

# int (Integer)

Stores whole numbers.

## Example

```python
age = 25
temperature = -10
population = 1000000

print(age)
print(type(age))
```

## Output

```python
25
<class 'int'>
```

---

# float

Stores decimal numbers.

## Example

```python
price = 99.99
pi = 3.14159

print(price)
print(type(price))
```

## Output

```python
99.99
<class 'float'>
```

---

# complex

Stores complex numbers.

Format:

```python
a + bj
```

## Example

```python
num = 2 + 3j

print(num)
print(type(num))
```

## Output

```python
(2+3j)
<class 'complex'>
```

---

# 2. Sequence Data Types

Sequence data types store multiple values in order.

## Types

- `str`
- `list`
- `tuple`
- `range`

---

# str (String)

Stores text data.

Strings are immutable.

## Example

```python
name = "Akash"

print(name)
print(type(name))
```

## Output

```python
Akash
<class 'str'>
```

---

## String Indexing

```python
text = "Python"

print(text[0])
print(text[-1])
```

## Output

```python
P
n
```

---

## Common String Methods

| Method | Purpose |
|---|---|
| `upper()` | Convert to uppercase |
| `lower()` | Convert to lowercase |
| `strip()` | Remove spaces |
| `replace()` | Replace text |
| `split()` | Split string |

## Example

```python
msg = "hello"

print(msg.upper())
```

---

# list

A list stores multiple items.

Lists are mutable.

## Example

```python
numbers = [10, 20, 30]

print(numbers)
print(type(numbers))
```

## Output

```python
[10, 20, 30]
<class 'list'>
```

---

## Access List Elements

```python
fruits = ["apple", "banana", "mango"]

print(fruits[0])
```

## Output

```python
apple
```

---

## Common List Methods

| Method | Purpose |
|---|---|
| `append()` | Add item |
| `insert()` | Insert item |
| `remove()` | Remove item |
| `pop()` | Remove by index |
| `sort()` | Sort list |

## Example

```python
nums = [1, 2, 3]

nums.append(4)

print(nums)
```

---

# tuple

Tuples are ordered collections.

Tuples are immutable.

## Example

```python
coordinates = (10, 20)

print(coordinates)
print(type(coordinates))
```

## Output

```python
(10, 20)
<class 'tuple'>
```

---

## Tuple Packing and Unpacking

```python
person = ("Akash", 25)

name, age = person

print(name)
print(age)
```

---

# range

Represents a sequence of numbers.

## Example

```python
numbers = range(5)

print(list(numbers))
```

## Output

```python
[0, 1, 2, 3, 4]
```

---

# 3. Set Data Types

Set stores unique values.

## Types

- `set`
- `frozenset`

---

# set

Unordered and mutable collection.

Duplicate values are not allowed.

## Example

```python
numbers = {1, 2, 3, 3, 4}

print(numbers)
```

## Output

```python
{1, 2, 3, 4}
```

---

## Common Set Methods

| Method | Purpose |
|---|---|
| `add()` | Add item |
| `remove()` | Remove item |
| `union()` | Combine sets |
| `intersection()` | Common elements |

## Example

```python
a = {1, 2}
b = {2, 3}

print(a.union(b))
```

---

# frozenset

Immutable version of set.

## Example

```python
data = frozenset([1, 2, 3])

print(data)
```

---

# 4. Mapping Data Type

# dict (Dictionary)

Stores data in key-value pairs.

Dictionary is mutable.

## Example

```python
student = {
    "name": "Akash",
    "age": 25,
    "course": "Python"
}

print(student)
```

---

## Access Dictionary Values

```python
print(student["name"])
```

## Output

```python
Akash
```

---

## Common Dictionary Methods

| Method | Purpose |
|---|---|
| `keys()` | Get keys |
| `values()` | Get values |
| `items()` | Get key-value pairs |
| `get()` | Get value safely |
| `update()` | Update dictionary |

## Example

```python
student["age"] = 26

print(student)
```

---

# 5. Boolean Data Type

# bool

Represents:

- `True`
- `False`

Used in conditions.

## Example

```python
is_logged_in = True

print(is_logged_in)
print(type(is_logged_in))
```

---

# Boolean Expressions

```python
print(10 > 5)
print(5 > 10)
```

## Output

```python
True
False
```

---

# 6. Binary Data Types

Used for binary data handling.

## Types

- `bytes`
- `bytearray`
- `memoryview`

---

# bytes

Immutable binary data.

## Example

```python
data = b"hello"

print(data)
```

---

# bytearray

Mutable binary data.

## Example

```python
data = bytearray([65, 66, 67])

print(data)
```

---

# memoryview

Provides memory access without copying data.

## Example

```python
data = memoryview(bytes(5))

print(data)
```

---

# 7. None Data Type

# NoneType

Represents absence of value.

## Example

```python
data = None

print(data)
print(type(data))
```

## Output

```python
None
<class 'NoneType'>
```

---

# Mutable vs Immutable Data Types

# Mutable

Can be changed after creation.

## Mutable Types

- list
- dict
- set
- bytearray

---

# Immutable

Cannot be changed after creation.

## Immutable Types

- int
- float
- complex
- str
- tuple
- frozenset
- bytes
- bool
- range

---

# Type Conversion

Python allows converting one type to another.

## Example

```python
age = "25"

converted_age = int(age)

print(converted_age)
print(type(converted_age))
```

---
